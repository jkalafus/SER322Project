Scripts and the requirements that they fulfill:
- creation_and_population is for creating and populating the books database. If it already exists, make sure to drop it before running.
- select_book_author is a select with 2 joins
- insert_genre_and_type is a insert adding genres and their genre types
- update_genre_nf_casing is a update altering the casing for the non-fiction genre type from "Non-fiction" to "Non-Fiction"
- insert_book_and_associations is an insert adding a book with its author, publisher, format and genre
